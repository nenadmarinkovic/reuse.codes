<Text>To remove the default blue underline from an anchor tag add: </Text>

```html
text-decoration: 'none'
```
<Text>Example usage with inline styling. Below is a live playground where you can edit the code. Try removing the styling and see what happens to the link below</Text>

```html react-live

<a href="somewhere-cool" style={{textDecoration: 'none'}}>I have no underline!</a>

```
