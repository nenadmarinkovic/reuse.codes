<Text>To create a new branch checkout the branch you want to base your work on </Text>

```bash
git checkout myOriginalBranch
```

<Text>Then its as simple as doing a pull to make sure you have the latest work and adding the -b tag with your new branches name </Text>

```bash
git pull
git checkout -b myNewBranch
```
