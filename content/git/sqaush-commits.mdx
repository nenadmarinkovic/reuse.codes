<Text>Before you merge your code its often a good idea to squash your commits. The first step is to find out how many commits have been made </Text>

```bash
git rev-list HEAD --count
```

<Text>Once we have the number of commits we can run the following command to squash them </Text>

```bash
git rebase -i HEAD~numberOfCommits
```
